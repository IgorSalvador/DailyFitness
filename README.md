# Daily Fitness

## Sobre o projeto

Projeto tem como bojetivo centralizar segmentos da área fitness em uma única plataforma oferecendo diversos serviços relacionados a saúde e bem estar. A proposta visa atender usuários que buscam a melhoria na saúde quanto os profissionais que ajudaram estes clientes comuns, oferecendo planos personalizados de treino e dieta, desafios diários e serviços de consultoria. Como resultado, espera-se um aumento na motivação nos usuários, contribuindo para um estilo de vida mais saudável.
